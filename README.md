## Purpose

Property listing app with Next.js SSG, Chakra UI, and Bayut API with mock fallback.

## Use Cases

- Real estate portfolio demo
- SSG with external API
- Responsive property cards

## Tech Stack

- Next.js 12
- Chakra UI
- Axios
- RapidAPI Bayut

## How to Run Locally

```bash
npm install && npm run dev
```
